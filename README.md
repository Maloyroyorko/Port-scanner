# Port-scanner
Port-scanner is a tool made in bash by Maloy Roy Orko to scan ports easily

Usage: 
chmod 777 port.sh

To Scan Specific Ports:
./port.sh 192.168.0.100 1 8080

To Scan All Posts:
./port.sh 192.168.0.100
